## StreamBoard - Stream Deck Audio Plugin
Stream Deck plugin for advanced soundbox features with support for local files and URLs, volume control, and multi-output management.

# Features
Audio playback of local MP3 files and remote URLs

Volume control (0-100%)

Loop/repeat mode

Specific audio device support (WIP)

Play/Stop Sound actions

# Installation

## From Stream Deck Store (Recommended) (Will soon be released)
Open Stream Deck
Go to Store → Search "StreamBoard"
Click Install

## Manual Installation

### Clone the repo
git clone https://github.com/bknadir/streamboard-streamdeck.git
cd streamboard-streamdeck

### Build
npm install
npm run build

I'll add release when I'll be satisfied with the features implemented
